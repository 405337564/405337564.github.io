内网渗透

3.内网渗透

3.1.Window信息收集

3.1.1.基本命令

  • 查询所有计算机名称dsquery computer
  • 查询配置systeminfo
  • 查看版本ver
  • 进程信息tasklist /svc
  • 查看所有环境变量set
  • 查看计划任务schtasks /QUERY /fo LIST /v
  • 查看安装驱动DRIVERQUERY

3.1.2.域信息

  • 获取当前组的计算机名net view
  • 当前所有域net view /domain
  • 查看域中的用户名dsquery user
  • 查看域组名称net group /domain
  • 查看域管理员net group "Domain Admins" /domain
  • 查看域控制器net group "Domain controllers"

3.1.3.用户信息

  • 查看用户net user
  • 查看在线用户qwinsta / query user
  • 查看当前计算机名，全名，用户名，系统版本，工作站域，登陆域net config Workstation

3.1.4.网络信息

  • 域控信息nltest /dclist:xx
